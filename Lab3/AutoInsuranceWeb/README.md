# AutoInsurance Web — ЛБ3

Front-end частина системи автострахування. Застосунок працює як SPA у браузері, має клієнтський та адміністративний інтерфейси, підтримує українську та англійську мови, форматування дат/валюти за регіоном, локалізоване сортування, LTR/RTL напрям інтерфейсу, експорт/імпорт і резервні копії даних.

## Запуск

Варіант без встановлення пакетів:
1. Відкрийте `index.html` у браузері.
2. Для стабільнішої роботи API краще запустити через Live Server у VS Code.

Варіант через Vite:

```bash
npm install
npm run dev
```

Після запуску відкрийте адресу, яку покаже термінал, наприклад `http://localhost:5173`.

## Сервер

За замовчуванням використовується Render URL:

```text
https://auto-insurance-server.onrender.com
```

Якщо сервер недоступний, застосунок не ламається: для демонстрації використовується локальний стан у `localStorage`, а всі рольові сценарії, заявки, резервні копії, імпорт/експорт і адміністрування працюють без зміни бекенда.

## Основні сценарії для захисту

1. Увійти як Driver і створити заявку про ДТП.
2. Перемкнути роль на Manager і обробити заявку: взяти в роботу, схвалити, позначити як виплачену.
3. Перемкнути роль на Admin і показати керування користувачами, даними, резервними копіями, експортом та імпортом.
4. Перемкнути мову UA/EN, регіон форматування, сортування й напрям інтерфейсу.
5. Показати телеметрію, створення події та визначення критичності.
